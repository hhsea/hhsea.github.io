yarn 的安装

使用 npm 安装 yarn：

如果 npm 使用的是默认设置，则不需要配置，可直接使用 yarn，如果更改过 npm 的目录，则需要在环境变量中指定 yarn 的安装目录：

  • 新建系统变量： yarn_bin，变量值为： D:\develop\nodejs\npm-global\bin。
  • 将 yarn_bin 加入 path的变量值中。
